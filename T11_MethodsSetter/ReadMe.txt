17. Методы set должны устанавливать значение из параметра
Метод Set устанавливает значение через свой параметр, 
плохая история когда вы под Set маскируете более информативный глагол. 
Т.е если вы через этот метод не устанавливаете значение 
(например метод SetMoney(int amount) который установит значение), то придумайте информативный глагол.
